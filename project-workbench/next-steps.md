# Next Steps

## 当前最高优先级

### 1. 真实图片替换与授权整理

当前剩余品牌车型已经有内容页面，但仍需要系统处理图片：

- 为每台车型确认真实授权图片
- 记录图片来源、作者、授权协议
- 优化图片尺寸和裁切
- 更新 `heroCredit` 与 `heroSourceHref`

建议顺序：

1. Lexus LFA / LFA Nürburgring Package
2. Lamborghini Huracan Performante / Aventador SVJ
3. Nissan GT-R R35 / GT-R NISMO / R34 GT-R
4. BMW M4 CSL / E30 M3
5. Ferrari 458 / 488 Pista / SF90

### 2. 重点车型拆独立文件

`brand-expansion-records.js` 适合快速打通页面，但长期维护时，重点车型应拆成独立文件。

优先拆：

- Lexus LFA
- Lamborghini Huracan Performante
- Nissan GT-R R35
- BMW M4 CSL
- Audi R8 LMS GT3
- Volkswagen ID.R

### 3. 事实核验

重点核验：

- 纽北圈速
- 圈速口径：20.6 km / 20.832 km
- 车手
- 年份
- 是否官方圈速、媒体测试或非正式测试
- 轮胎条件
- 动力参数

不要把待核实数据写成确定事实。

### 4. 移动端专项检查

需要重点看：

- 首页地图
- 品牌详情页车型卡片
- 车型详情页长标题
- 图片 hero 裁切
- 资料来源卡片

### 5. 内容风格统一

当前 Porsche / Mercedes-AMG 的完成度最高。

后续品牌文案应向这两组靠拢：

- 不写百科条目
- 不重复“从 A 到 B”的句式
- 不只讲“很快”
- 每台车要有明确角色
- 保持汽车杂志与赛道展览语气
